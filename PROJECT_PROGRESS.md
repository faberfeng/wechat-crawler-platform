## 🏗️ 技术栈

### 后端
- **框架：** FastAPI 0.109.0
- **数据库：** SQLite + SQLAlchemy 2.0.25
- **认证：** JWT (python-jose)
- **密码：** passlib + bcrypt/SHA256
- **运行：** Uvicorn

### 前端
- **框架：** Vue 3 (Composition API)
- **UI：** Element Plus
- **状态：** Pinia
- **路由：** Vue Router
- **构建：** Vite

### 附加功能
- **公网访问：** Cloudflare Tunnel
- **自动化：** Shell 脚本
- **测试：** API 自动化测试
