## 📊 API 统计

### 认证相关（6 个）
- POST /register - 用户注册
- POST /login - 用户登录
- POST /logout - 用户登出
- GET /auth/me - 获取当前用户
- PATCH /auth/me - 更新用户信息
- PATCH /auth/me/password - 修改密码
- DELETE /auth/me - 删除账号

### 用户管理（5 个）
- GET /users/ - 获取用户列表
- GET /users/{id} - 获取用户详情
- POST /users/ - 创建用户
- PATCH /users/{id} - 更新用户
- DELETE /users/{id} - 删除用户

### 公众号管理（7 个）
- POST /accounts - 添加公众号
- GET /accounts - 获取列表
- GET /accounts/{id} - 获取详情
- PUT /accounts/{id} - 更新公众号
- DELETE /accounts/{id} - 删除公众号
- POST /accounts/{id}/crawl - 触发抓取
- PUT /accounts/{id}/toggle - 切换状态

### 文章管理（5 个）
- GET /articles - 获取列表
- GET /articles/{id} - 获取详情
- GET /articles/{id}/markdown - 获取内容
- DELETE /articles/{id} - 删除文章
- GET /articles/stats/summary - 统计数据

### 任务管理（3 个）
- GET /tasks - 获取任务列表
- GET /tasks/{id} - 获取任务详情
- POST /tasks/health-check - 健康检查

### 文件管理（7 个）
- POST /files/upload - 上传文件
- GET /files/ - 获取列表
- GET /files/{id} - 获取文件信息
- GET /files/{id}/download - 下载文件
- GET /files/{id}/preview - 预览文件
- DELETE /files/{id} - 删除文件
- GET /files/stats/summary - 获取统计信息


### 前端启动
```bash
cd frontend
npm install
npm run dev
```
