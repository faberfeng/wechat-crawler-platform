### 2. 后端 API 接口 ✅

**文件：** `backend/app/api/v1/files.py`

#### 已实现的函数：

#### 1. 文件上传
**路径：** `POST /api/v1/files/upload`

**功能：**
- 文件大小验证（最大 50MB）
- 文件类型验证（支持图片、文档、压缩包）
- 生成唯一文件名
- 按用户和日期组织目录
- 保存文件到本地存储
- 数据库记录元信息
- 返回文件信息

**允许的文件类型：**
- **图片：** .jpg, .jpeg, .png, .gif, .webp, .bmp
- **文档：** .pdf, .doc, .docx, .txt, .md, .xls, .xlsx, .ppt, .pptx
- **压缩包：** .zip, .rar, .7z, .tar, .gz

**请求示例：**
```bash
curl -X POST "http://localhost:8000/api/v1/files/upload" \
  -H "Authorization: Bearer <token>" \
  -F "file=@/path/to/file.pdf"
```

**响应示例：**
```json
{
  "id": 1,
  "filename": "original.pdf",
  "file_size_formatted": "1.5 MB",
  "category": "document",
  "mime_type": "application/pdf",
  "uploaded_at": "2026-03-26T12:00:00"
}
```

---

#### 2. 文件列表
**路径：** `GET /api/v1/files/`

**功能：**
- 获取用户的所有文件
- 支持分页（skip, limit）
- 按类别过滤
- 按上传时间降序排序

**参数：**
- `skip` - 跳过数量（默认 0）
- `limit` - 返回数量（默认 20）
- `category` - 文件类别（image/document/archive）

---

#### 3. 文件信息
**路径：** `GET /api/v1/files/{file_id}`

**功能：**
- 获取单个文件的详细信息
- 包含格式化的文件大小
- 自动识别文件类别

---

#### 4. 文件下载
**路径：** `GET /api/v1/files/{file_id}/download`

**功能：**
- 下载文件
- 使用原始文件名
- 正确的 MIME 类型
- 只能下载自己的文件

---

#### 5. 文件预览
**路径：** `GET /api/v1/files/{file_id}/preview`

**功能：**
- 在线预览文件（仅支持图片）
- 返回二进制内容
- 正确的 Content-Type
- 非图片返回错误

---

#### 6. 文件删除
**路径：** `DELETE /api/v1/files/{file_id}`

**功能：**
- 删除物理文件
- 删除数据库记录
- 验证文件所有权
- 只能删除自己的文件

---

#### 7. 文件统计
**路径：** `GET /api/v1/files/stats/summary`

**功能：**
- 文件总数
- 总文件大小
- 按类别统计（数量和大小）
- 格式化的大小显示

**响应示例：**
```json
{
  "total_count": 10,
  "total_size": 52428800,
  "total_size_formatted": "50.00 MB",
  "categories": {
    "image": {"count": 5, "size": 25600000, "size_formatted": "24.41 MB"},
    "document": {"count": 4, "size": 26828800, "size_formatted": "25.59 MB"},
    "archive": {"count": 1, "size": 1024, "size_formatted": "1.00 KB"}
  }
}
```

## 🔧 需要修复的问题

### Python 版本兼容性

**问题描述：**
系统 Python 3.9 不支持 `str | None` 类型注解语法。

**错误信息：**
```
TypeError: unsupported operand type(s) for |: 'type' and 'NoneType'
```

**位置：**
- `backend/app/api/v1/auth.py` 第 76 行

**解决方案：**
将 `str | None` 改为 `Optional[str]`
