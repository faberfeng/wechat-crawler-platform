## 🐛 常见问题排查

### 问题 1：无法访问公网地址

**症状：**
- 访问 https://fwb-wechat.loca.lt 显示空白或错误

**解决方案：**
1. 检查 Cloudflare Tunnel 是否运行
2. 重启服务：`./start.sh`
3. 查看浏览器控制台错误

---

### 问题 2：登录后立即跳转回登录页

**症状：**
- 登录成功后，又自动跳转到登录页面

**可能原因：**
- Token 未正确保存到 localStorage
- Token 格式不正确
- 响应拦截器错误

**解决方案：**
1. 检查 Console 错误日志
2. 检查 localStorage 中是否有 token
3. 检查 Network 标签中的 API 响应

---

### 问题 3：管理员看不到"用户管理"菜单

**症状：**
- 登录管理员账号，但没有"用户管理"菜单

**可能原因：**
- 用户角色不是 'admin'
- 路由守卫判断错误

**解决方案：**
1. 检查用户信息中的 role 字段
2. 打开 Console，输入：
   ```javascript
   JSON.parse(localStorage.getItem('user')).role
   ```
3. 应该显示 'admin'

---

### 问题 4：API 请求失败

**症状：**
- Network 标签显示 API 请求失败（400/401/500 错误）

**解决方案：**
1. 检查后端服务是否运行
2. 检查 CORS 配置
3. 查看响应体中的错误信息
4. 检查请求头是否包含 Authorization
