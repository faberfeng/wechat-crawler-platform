# 快速添加链接工具 - 使用指南

## 📋 概述

两个快速添加微信文章链接的工具：
- **`quick_add.sh`** - Bash 脚本版本（Mac/Linux）
- **`quick_add.py`** - Python 版本（跨平台）

推荐使用 Python 版本，功能更全面。

---

## 🚀 快速开始

### 最简单的用法

```bash
cd ~/Desktop/wechat-crawler-platform

# 1. 复制微信文章链接（从微信 App）

# 2. 运行脚本（自动从剪贴板读取）
python quick_add.py --paste --run
```

就这么简单！脚本会：
- ✅ 自动从剪贴板读取链接
- ✅ 添加到配置文件
- ✅ 立即运行抓取
- ✅ 保存到数据库

## 📖 详细用法

### Python 版本（推荐）

#### 方式一：从剪贴板（最快）

```bash
# 添加并立即运行抓取
python quick_add.py --paste --run

# 仅添加不运行
python quick_add.py --paste
```

**工作流：**
1. 打开微信 App
2. 进入"上海静安"公众号
3. 长按文章 → 链接 → 复制
4. 运行 `python quick_add.py --paste --run`
5. 完成！

---

#### 方式二：指定链接

```bash
# 添加单个链接
python quick_add.py --link 'https://mp.weixin.qq.com/s/ZKXjKURWMMStlij6OhUbiA'

# 添加多个链接（逗号分隔）
python quick_add.py --links 'url1,url2,url3'

# 添加并运行
python quick_add.py --link 'https://mp.weixin.qq.com/s/xxx' --run
```

---

#### 方式三：从文件读取

```bash
# 创建文件，每行一个链接
cat > links.txt << 'EOF'
https://mp.weixin.qq.com/s/文章1
https://mp.weixin.qq.com/s/文章2
https://mp.weixin.qq.com/s/文章3
EOF

# 从文件读取并添加
python quick_add.py --file links.txt --run
```

---

#### 方式四：交互式输入

```bash
# 进入交互式模式，手动输入链接
python quick_add.py --interactive
```

会提示：
```
请输入文章链接（多个链接用空格或逗号分隔）：
提示: 也可以先复制链接，然后使用 --paste 选项

> https://mp.weixin.qq.com/s/文章1 https://mp.weixin.qq.com/s/文章2
```

---

#### 方式五：管理配置

```bash
# 显示当前配置
python quick_add.py --show

# 清空配置（会自动备份）
python quick_add.py --clear

# 指定公众号 ID（默认为 1）
python quick_add.py --link 'xxx' --account 2
```

---

### Bash 版本

## ❓ 常见问题

### 1. 剪贴板读取失败

**症状：** 显示"无法访问剪贴板"

**解决：**
- 确保已复制链接
- Mac 需要终端有权限访问剪贴板
- 或改用 `--link` 选项手动输入

---

### 2. 链接格式不正确

**症状：** 警告"链接格式可能不正确"

**正确格式：**
```
https://mp.weixin.qq.com/s/ZKXjKURWMMStlij6OhUbiA
```

**不正确格式：**
```
https://mp.weixin.qq.com/s/链接标题  # 带标题
http://mp.weixin.qq.cn/s/xxx  # 不是 .qq.com
https://mp.weixin.qq.com/mp/xxx  # 不是 /s/ 开头
```

---

### 3. 链接数量过多

**症状：** 警告"链接数量过多，仅保留最新的100个"

**说明：** 为避免配置文件过大，系统限制最多保存100个链接

**解决：** 如果需要更多，可以：
- 分多次添加
- 清空旧链接：`python quick_add.py --clear`

---

### 4. 运行后没有抓取

**检查：**
```bash
# 1. 查看日志
tail -100 /tmp/wechat_crawl.log

# 2. 检查配置
python quick_add.py --show

# 3. 手动运行调试
cd ~/Desktop/wechat-crawler-platform
python crawl_articles.py --account-id 1
```

### 技巧 3：定期维护

```bash
# 创建维护脚本
cat > maintenance.sh << 'EOF'
#!/bin/bash

echo "配置文件大小："
ls -lh config/article_urls.txt

echo ""
echo "当前链接数量："
python quick_add.py --show | grep -o "总链接数: [0-9]*"

echo ""
read -p "是否清空旧配置？[y/N]: " confirm

if [[ "$confirm" =~ ^[yY]$ ]]; then
    python quick_add.py --clear
    echo "✓ 配置已清空"
else
    echo "已取消"
fi

EOF
chmod +x maintenance.sh
```
