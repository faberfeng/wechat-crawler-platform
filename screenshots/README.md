# Screenshots

此目录用于存放项目截图。

## 建议添加的截图

1. **dashboard.png** - 仪表盘界面
2. **accounts.png** - 公众号管理界面
3. **articles.png** - 文章列表界面
4. **users.png** - 用户管理界面
5. **files.png** - 文件管理界面
6. **login.png** - 登录界面
7. **article-detail.png** - 文章详情界面

## 如何添加截图

1. 在浏览器中打开应用
2. 使用截图工具（macOS: Cmd+Shift+4）
3. 保存为 PNG 格式到本目录
4. 更新 README.md 中的截图链接

注意：为了 GitHub 显示效果，建议截图宽度不超过 1200px。
